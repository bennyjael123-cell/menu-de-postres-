# 🍰 Sweet Desserts

**Sweet Desserts** es un menú digital de postres creado para mostrar diferentes opciones dulces de manera visual y moderna.

Nuestro objetivo es ofrecer una experiencia simple y atractiva para que los clientes puedan explorar nuestros postres favoritos.

## 🍩 Menú

Entre nuestros postres encontrarás:

* Pastel de chocolate
* Cheesecake de fresa
* Helado artesanal
* Donas glaseadas
* Waffles dulces
* Brownies

## 💻 Tecnología

Este menú fue desarrollado usando:

* HTML
* CSS
* Imágenes de internet (Unsplash)

## 🎯 Objetivo

Crear un menú digital fácil de usar que permita a los clientes ver los postres disponibles y sus precios.
